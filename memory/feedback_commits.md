name: feedback-commits
description: Préférences de l'utilisateur pour les messages de commit Git
metadata:
  type: feedback
---

Ne pas inclure de ligne `Co-Authored-By: Claude ...` dans les messages de commit.

**Why:** L'utilisateur ne veut pas que le nom de Claude apparaisse dans l'historique Git.

**How to apply:** Écrire les messages de commit sans aucune ligne `Co-Authored-By` ou signature automatique.
